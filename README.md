## 1. 介绍

上海市遥感影像与带高度的建筑物数据集

图片分辨率为**512px * 512px**，格式为`GeoTIFF`，文件后缀为`.tif`，带有坐标信息，坐标系为`EPSG3857`

6000多个样本，解压缩后数据量为9个G左右，建筑物覆盖率在10％以上，遥感影像来源Google地图

文件夹`samples`下为示例数据，共14组，文件名字为`height_<x>_<y>`和`google_<x>_<y>`相对应

下载地址为：
